## エラーコード参考
- BSEC_E_SU_SAMPLERATELIMITS (-12): サンプルレート制限エラー
- 一部のセンサーは特定のサンプルレート（ULP/LP/CONT）でのみ利用可能

## サンプルレート
- BSEC_SAMPLE_RATE_ULP: 5分間隔（超低消費電力）
- BSEC_SAMPLE_RATE_LP: 3秒間隔（低消費電力）- 現在使用中
- BSEC_SAMPLE_RATE_CONT: 1秒間隔（連続測定）
